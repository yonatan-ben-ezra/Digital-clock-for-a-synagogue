# שעון דיגיטלי לבית הכנסת

אתר אינטרנט המציג שעון דיגיטלי גדול עם תאריך לועזי ועברי, זמני זריחה והשקיעה, וטיימר אוטומטי.

## תכונות

- 🕐 **שעון דיגיטלי** - שעה, דקה, שנייה בעיצוב גדול וברור
- 📅 **תאריך לועזי ועברי** - תאריך בשתי השפות
- 🌅 **זמני זריחה והשקיעה** - לפי המיקום שנקבע
- ⏱️ **טיימר אוטומטי** - מופיע X דקות לפני הנץ
- ⚙️ **הגדרות ניתנות לשינוי** - בחר כמה דקות לפני הנץ להפעיל טיימר

## התחלה

1. פתח את `index.html` בדפדפן
2. או העלה את הקבצים לשרת ואפשר גישה דרך כתובת אינטרנט

## הגדרות

לחץ על כפתור ⚙️ כדי:
- לשנות את זמן הטיימר (דקות לפני הנץ)
- לעדכן את המיקום (כברירת מחדל: זאב פלק 18, ירושלים)

## קבצים

- `index.html` - העמוד הראשי
- `style.css` - עיצוב וסגנון
- `app.js` - לוגיקה האפליקציה
- `config.js` - הגדרות ברירת מחדל

## טכנולוגיה

- HTML5
- CSS3
- JavaScript (Vanilla)
